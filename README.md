# 🚀 UNIP-ADS-FRONTEND-2026
Repositório dedicado ao armazenamento de projetos, exercícios e estudos práticos desenvolvidos durante o curso de Análise e Desenvolvimento de Sistemas (ADS) na UNIP, com foco total em tecnologias Front-end.

![UNIP](https://img.shields.io/badge/UNIP-ADS-blue?style=for-the-badge)

# 💻 Sobre o Repositório
Este espaço serve como meu laboratório de desenvolvimento. Aqui, aplico os conceitos de UX/UI aprendidos em prototipagem para criar interfaces que não são apenas bonitas, mas funcionais e acessíveis. O objetivo é transitar do conhecimento acadêmico para soluções reais de software.

# 🛠️ Tecnologias e Conceitos Explorados
HTML5 Semântico: Estruturação de páginas focada em SEO e acessibilidade.

CSS3 Avançado: Uso de Flexbox, CSS Grid e animações para layouts modernos.

JavaScript (ES6+): Manipulação de DOM, consumo de APIs e lógica de programação para tornar as páginas interativas.

Bootstrap 5: Agilidade no desenvolvimento de interfaces responsivas e componentes mobile-first.

Mobile First & Responsividade: Garantindo que cada projeto funcione perfeitamente em computadores, notebooks e smartphones.

# 📂 Estrutura de Projetos
Abaixo, alguns dos desafios e projetos práticos que você encontrará por aqui:

Formulários Inteligentes: Implementação de formulários de feedback e reclamação com validações em JS (baseados em wireframes do draw.io).

Landing Pages: Páginas de alta conversão focadas em serviços de tecnologia.

Dashboards: Interfaces de controle e visualização de dados.
